# RvB.Collections
Various set based data structures:

- BitSet32: efficiently manage a set numbers (0-31)
- BitSet64: efficiently manage a set numbers (0-63)
- DeQueue: a double sided queue, allowing items to be enqueued and dequeued from the front and the back.
- ValuArray: an array with value like characteristics. Equality is based on content and not on reference.
- LinkedListArray: a linked list implementation based on an underlying array.
- UniquePriorityQueue: ...
- FibonacciPriorityQueue: ...
- Sets: allows for the generation of subsets
- OrderedList: ...
